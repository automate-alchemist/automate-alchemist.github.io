<b>8th January - Selenium: Browser Navigation and Assertions</b>

* <b>Learning:</b> Explored browser navigation and assertions in Selenium.

<b>7th January - Selenium: Handling Waits</b>

* <b>Learning:</b> Learned about handling waits in Selenium.

* <b>Description:</b> I explored the different types of waits in Selenium: implicit, explicit, and fluent waits. Each type helps in managing synchronization issues between the browser’s actions and the script’s execution.

* <b>Key Takeaway:</b> Managing waits properly is crucial for ensuring that automation scripts run smoothly without errors due to page loading times or dynamic content.

* <b>Personal Reflection:</b> This was an important topic, as I’ve encountered issues in previous projects due to improper waits. Now, I have the tools to fix those issues and improve test reliability.
